char c = 'A'와
char c = "A"의 차이가 뭐야?
GTP
char c = 'A'는 문자형 변수 c에 문자 A를 할당하는 것이고,
char c = "A"는 문자열 "A"를 문자형 변수 c에 할당하려는 것입니다.
자바에서 문자는 작은따옴표('')로 감싸서 표현하며,
문자열은 큰따옴표("")로 감싸서 표현합니다.
따라서 char c = "A"는 컴파일 오류가 발생합니다.
아 그러니까 char은 문자형변수, 즉 1개의 문자만 가능하고 그때 1개의 문자일땐 작은 따옴표를 쓰는구나?
GTP
맞습니다. char 타입은 1개의 문자만을 저장하는 데이터 타입으로,
작은 따옴표(' ')를 사용합니다.
큰 따옴표(" ")는 문자열을 나타내는 데이터 타입인 String 타입을 사용합니다.
